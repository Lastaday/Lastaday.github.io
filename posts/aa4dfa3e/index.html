Hexo搭建Blog教程

发表于2022-05-17|更新于2022-05-17|网站搭建
|阅读量:

Hexo是一个快速、简洁且高效的博客框架。Hexo支持Markdown语言来编辑文章，普通用户可使用Markdown 编辑器Typora编辑内容，hexo通过其自带插件将md文件生成静态网页内容。

安装node.js

hexo是一个基于nodejs 开发的静态博客网站生成器，在安装hexo之前，需要先安装node。node下载地址。如果在本地需要管理多个node包，可以使用nvm来下载管理node。nvm可使用简单的命令来安装、查看、切换当前的node版本。

hexo命令

常见命令
  • hexo new “postName” #新建文章
  • hexo new page “pageName” #新建页面
  • hexo generate #生成静态页面至public目录
  • hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
  • hexo deploy #部署到GitHub
  • hexo help # 查看帮助
  • hexo version #查看Hexo的版本
缩写
  • hexo n == hexo new
  • hexo g == hexo generate
  • hexo s == hexo server
  • hexo d == hexo deploy
组合命令
  • hexo s -g #生成并本地预览
  • hexo d -g #生成并上传

更换主题，美化页面样式

hexo默认使用的主题是landscape，看的碜的慌，对不对。选择一套好看的主题吧。在github上搜索hexo themes。按照most star排序。选择一套你喜欢的主题吧。next，yilia，matery，butterfly都是用的比较多的主题。我选择了butterfly，理由是界面简洁，文档丰富。

使用abbrlink转换url，便于搜索引擎优化

在使用hexo new “post名称” 命令生成post文章.md时，如果post名称名称是中文。那么在访问时url地址也是带中文的，这样对搜索引擎并不友好。使用hexo-abbrlink可以将地址变成随机英文字符串。
